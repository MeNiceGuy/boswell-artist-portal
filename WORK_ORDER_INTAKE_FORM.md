# Work Order Intake Form

Use this as the field list for Tally, Fillout, Typeform, Airtable, Jotform, or a custom portal.

## Artist Information

- Legal name
- Artist/stage name
- Email
- Phone
- Country
- State/province
- Preferred contact method
- Are you 18 or older?
- If under 18, parent/guardian contact

## Package Selection

- Royalty Account Setup
- Single Song Registration Work Order
- Release Pack
- Catalog Cleanup Sprint
- Monthly Royalty Admin Desk
- Rush add-on

## Portal Selection

- BMI
- ASCAP
- The MLC
- SoundExchange
- Not sure / need review

## Current Account Status

- I already have BMI access
- I already have ASCAP access
- I already have The MLC access
- I already have SoundExchange access
- I need account setup support
- I am not sure

## Song or Release Details

- Work order type: single song, EP, album, old catalog cleanup
- Song title
- Alternate title
- Recording title, if different
- Primary artist
- Featured artists
- Writers
- Producers
- Master owner
- Publisher or self-administered status
- Release date
- Distributor
- UPC
- ISRC
- DSP/store links
- Lyrics available?
- Explicit content?
- Is this already released?
- Is this already registered anywhere?

## Splits and Ownership

- Writer split percentages
- Publisher split percentages
- Master ownership percentages
- Featured artist agreement status
- Producer agreement status
- Any unresolved split disputes?
- Any samples or interpolations?
- Any cover song involved?
- Upload split sheet or agreement, if available

## Portal Task Request

- Register composition with selected PRO
- Register or claim composition with The MLC
- Register or claim recording with SoundExchange
- Review existing portal entries
- Correct missing metadata
- Prepare missing-information report only

## Credentials and Access

Preferred secure access method:

- Artist will log in during a screen-share session
- Artist will create limited access if portal allows it
- Artist will provide credentials through an approved password manager
- Artist only wants instructions, not done-for-you login support

Do not collect passwords through a normal text field or email.

## Authorization

Required checkbox:

I authorize Boswell Consulting Group to perform administrative royalty-registration support for the songs, recordings, and portals I selected in this work order. I understand this is administrative support only and does not guarantee royalties, income, approval, collection, legal ownership, tax treatment, or platform acceptance.

Required checkbox:

I confirm that the information I submitted is accurate to the best of my knowledge and that I have the right to request administrative support for the listed songs and recordings.

Required checkbox:

I understand that unresolved splits, missing data, samples, covers, ownership disputes, or incomplete portal access may delay or block completion.
